3. **–í–∏–≤–µ–¥–µ–Ω–Ω—è —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó**: –ü–æ–∫–∞–∑—É—î –∫–∞—Ä—Ç–∏ –æ–±–æ—Ö –≥—Ä–∞–≤—Ü—ñ–≤

5. **–û–Ω–æ–≤–ª–µ–Ω–Ω—è —Ä–∞—Ö—É–Ω–∫—É**:

7. **–†–µ–∫—É—Ä—Å–∏–≤–Ω–∏–π –≤–∏–∫–ª–∏–∫**: –í–∏–∫–ª–∏–∫–∞—î `play_game()` –∑–Ω–æ–≤—É –¥–ª—è –Ω–∞—Å—Ç—É–ø–Ω–æ–≥–æ —Ä–∞—É–Ω–¥—É

##### `fight(card1, card2)` (—Å—Ç–∞—Ç–∏—á–Ω–∏–π –º–µ—Ç–æ–¥)
–í–∏–∑–Ω–∞—á–∞—î –ø–µ—Ä–µ–º–æ–∂—Ü—è –±–æ—é –º—ñ–∂ –¥–≤–æ–º–∞ –∫–∞—Ä—Ç–∞–º–∏.

1. –†–æ–∑—Ä–∞—Ö–æ–≤—É—î –ø–æ—à–∫–æ–¥–∂–µ–Ω–Ω—è –¥–ª—è –∫–∞—Ä—Ç–∏ 1: `res_1 = card1.attack - card2.defense`
2. –†–æ–∑—Ä–∞—Ö–æ–≤—É—î –ø–æ—à–∫–æ–¥–∂–µ–Ω–Ω—è –¥–ª—è –∫–∞—Ä—Ç–∏ 2: `res_2 = card2.attack - card1.defense`

**–ü—Ä–∏–∫–ª–∞–¥ –±–æ—é**:

# res_1 = 8 - 3 = 5
